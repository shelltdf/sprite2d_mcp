# 系统设计

## 子系统

1. **UI 壳**：固定布局托管菜单栏、工具栏、三栏主区、状态栏；命令统一经 Pinia actions 或组合式函数触发。
2. **精灵编辑子系统**：列表与属性面板共享同一选中状态；属性变更回流到列表显示与画布几何。
3. **画布子系统**：读取世界坐标下的精灵矩形与视图矩阵，处理指针事件（平移、缩放、选中）。
4. **编译子系统**：根据当前精灵矩形集合计算图集尺寸（最小可行：包围盒或简单行排布），绘制到离屏 Canvas，导出 PNG + JSON。

## 与物理阶段对应

- 可交付物为前端 SPA，对应 `02-physical/sprite-sheet-compiler-gui/`。
- 对外行为以该目标下 `spec.md` 为字段级权威。

## 依赖关系

```
UI 壳 → 精灵编辑 / 画布 / 编译
精灵编辑 ↔ 画布（选中、几何）
编译 ← 精灵编辑（只读快照）
```
