# 系统设计

## 子系统

1. **UI 壳**：固定布局托管菜单栏、工具栏、三栏主区、底部时间线、状态栏；命令经 Pinia **`project`** / **`ui`** actions 触发；工具栏为**仅图标**按钮。
2. **精灵编辑子系统**：左侧列表与属性面板、画布共享 **`selectedIds`** 与动画选中态；属性变更经 store 回流列表与画布几何；**frame/pivot 整数**与几何钳制见 `spriteGeometry.js`。
3. **画布子系统**：世界坐标下的精灵矩形与 `view` 矩阵；指针事件（中键/空格+左键平移、滚轮缩放、框选、**先选中再拖动**、单选调整尺寸）；向 store 同步 **`canvasPointerOver` / 世界坐标 / 视口 CSS 尺寸** 以支持「添加精灵」落点与视口中心。
4. **动画子系统**：`animations[]`、时间线条带与播放；属性面板内**独立预览**轮播帧切片。
5. **编译子系统**：行排布打包画布、底图切片或色块占位、下载 `atlas.png` + `atlas.json`。
6. **互操作子系统**：`atlasImport.js` 导入第三方描述 + 位图；`exportAtlasMetadata.js` 导出无像素 JSON/XML；工程文件 v2 路径引用。

## 与物理阶段对应

- 可交付物为前端 SPA，对应 `02-physical/sprite-sheet-compiler-gui/`。
- 对外行为以该目标下 **`spec.md`** 为字段与交互权威。

## 依赖关系

```
UI 壳 → 精灵编辑 / 画布 / 动画展示 / 编译 / 互操作
精灵编辑 ↔ 画布（选中、几何、聚焦事件）
编译 ← 精灵编辑（只读快照）
互操作 ↔ 精灵编辑（导入替换列表与底图；导出只读快照）
```
