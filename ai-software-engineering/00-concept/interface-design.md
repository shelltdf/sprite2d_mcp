# 接口设计（程序间）

本工具以**本地 GUI** 为主；对外的程序间接口体现为**导出文件格式**（非 HTTP/RPC）。

## `atlas.json`（导出）

| 字段 | 类型 | 说明 |
|------|------|------|
| `version` | number | 格式版本，当前为 `1` |
| `image` | string | 图集文件名，如 `atlas.png` |
| `sprites` | array | 每项含 `name`, `x`, `y`, `w`, `h`（图集内像素坐标） |

## 与引擎对接

游戏引擎侧只需读取 PNG + `atlas.json` 即可切片；具体坐标系（Y 向上/向下）由消费方约定，导出为**画布像素坐标，原点在左上**。

> 说明：GUI 菜单、快捷键、窗口消息等**不属于**本文件范围，见 `product-design.md` 与逻辑阶段 UI 详细设计。
