# 软件设计

## 总体结构

- **表现层**：Vue 3 单页应用；菜单栏、工具栏、左 Dock、画布、时间线 Dock、属性面板、状态栏、右键菜单。
- **状态层**：Pinia **`project`**（工程、精灵、动画、视图、精灵表引用、画布指针/视口提示用于添加精灵）；Pinia **`ui`**（主题、画布上精灵框/底图显示开关）。
- **画布层**：Canvas 2D 渲染底图、精灵矩形、选中、调整手柄、框选；指针事件驱动平移、缩放、选中/拖动策略。
- **编译层**：`compileAtlas.js` 行布局、底图切片、`atlas.png` + `atlas.json`；`exportAtlasMetadata.js` 元数据 JSON/XML；`atlasImport.js` 多格式解析。

## 模块划分

| 模块 | 职责 |
|------|------|
| `App` / 布局壳 | 网格分区，挂载各面板与全局热键 |
| 菜单/工具栏 | 命令分发，调用 `project`/`ui` store |
| `LeftDock` | 精灵表设置、精灵表、动画列表；列表双击聚焦画布 |
| 画布 | 绘制与命中测试；同步画布指针世界坐标与视口尺寸至 store |
| 属性面板 | 精灵字段与预览；动画帧编辑与**动画预览**定时器 |
| 时间线 Dock | 帧条带与播放（与属性预览独立） |
| 编译/导入/元数据导出 | 见 `src/lib/*.js` |

## 与 `foo`/helloworld 说明

本仓库产品为精灵表工具，**不**采用 init 规则中的 helloworld + foo 示例；物理阶段以 `sprite-sheet-compiler-gui` 目标为准。
