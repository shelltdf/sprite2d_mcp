# 产品设计：Sprite2D 精灵表编译工具

## 愿景

为 2D 游戏与工具链提供一款**桌面风格**的精灵表（Sprite Sheet）编辑与编译工具：在可视化画布上摆放精灵区域，管理精灵元数据，并导出图集图像与描述文件。

## 目标用户

- 独立游戏开发者、技术美术
- 需要在 Windows 风格工作界面下快速打包散图或调整图集布局的人员

## 成功标准

- 主界面具备**菜单栏、工具栏、状态栏**，布局符合常见 Windows 应用习惯。
- **左侧**可浏览与管理精灵对象列表；**中央** 2D 画布展示图集与选区；**右侧**预览并编辑当前选中精灵的属性。
- 可执行基础的**编译/导出**流程（图集 PNG + JSON 元数据），并在状态栏反馈进度与消息。

## 界面结构（自上而下）

1. **菜单栏**：文件（新建/打开/保存/导出）、编辑（撤销占位）、视图（缩放占位）、帮助。
2. **工具栏**：常用快捷操作（新建工程、添加精灵、编译导出、适应画布等）。
3. **主工作区**（横向三栏）：
   - **左**：精灵对象表（列表/表格），支持选择与多选概念预留。
   - **中**：2D 画布（图集预览、矩形框选、平移/缩放入口）。
   - **右**：精灵属性预览区（名称、源矩形、在图集中的位置与尺寸等）。
4. **状态栏**：当前操作提示、选中项摘要、可选缩放比例显示。

## 非目标（当前阶段）

- 完整图像算法（复杂打包、旋转块、多图集自动拆分）可作为后续迭代。
- 非 Windows 平台的原生壳（Electron/Tauri）可选，当前以浏览器内 Win 风格视觉为主。
