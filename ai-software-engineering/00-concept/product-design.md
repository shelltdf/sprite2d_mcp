# 产品设计：Sprite2D 精灵表编译工具

## 愿景

为 2D 游戏与工具链提供一款**桌面风格**的精灵表（Sprite Sheet）编辑与编译工具：在可视化画布上摆放精灵区域，管理精灵元数据（含锚点、内边界与动画序列），并导出图集图像与描述文件；支持从常见第三方工具图集描述导入。

## 目标用户

- 独立游戏开发者、技术美术
- 需要在 Windows 风格工作界面下快速打包散图、调整图集布局或与 TexturePacker/Phaser 等管线互操作的人员

## 成功标准

- 主界面具备**菜单栏、工具栏（图标+悬停说明）、状态栏**，布局符合常见 Windows 应用习惯。
- **左侧 Dock**：精灵图、精灵对象表、动画序列；**中央**画布 + **下方**动画时间线；**右侧**属性（精灵或当前动画），含精灵裁剪预览、**辅助线开关**，以及动画的**独立预览播放**。
- **工程**：新建/打开/**保存/另存**（`.spriteproj.json`，精灵表仅存路径引用 v2）。
- **第三方图集**：导入描述（JSON/XML）+ 位图；导出打包图集（PNG+JSON）及**仅元数据** JSON/XML。
- 状态栏反馈操作结果与错误（多语言）。

## 界面结构（自上而下）

1. **菜单栏**：文件（新建/打开/保存/**另存**/导入与导出第三方图集及元数据）、精灵图、精灵、动画、视图、主题、语言、帮助。
2. **工具栏**：图标按钮——新建、添加精灵、设置精灵表、导入/导出第三方图集、适应窗口、精灵与底图显示开关等。
3. **主工作区**（横向三栏）：
   - **左**：Dock 折叠块（精灵表缩略图、精灵表、动画列表）。
   - **中**：2D 画布（底图、精灵框、锚点、内边界、框选、平移/缩放）。
   - **右**：属性（精灵数值与预览，或动画帧列表与动画预览）。
4. **底部时间线**：当前动画帧条带与播放（与属性区动画预览相互独立）。
5. **状态栏**：消息、缩放与选中摘要。

## 非目标（当前阶段）

- 复杂旋转块、多图集自动拆分等高级装箱可作为后续迭代。
- 非浏览器壳（Electron/Tauri）可选；当前以浏览器内 Win 风格视觉为主。
