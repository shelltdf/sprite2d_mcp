# 用户说明书：Sprite2D 精灵表编译工具

## 环境前提

- 现代浏览器（Chromium / Edge / Firefox 近期版本）。
- 通过开发服务器或构建后的静态文件打开应用。

## 界面说明

- **菜单栏**：文件操作与视图缩放；**工具栏**：同等快捷入口。
- **左侧**：精灵列表，点击可选中。
- **中央**：画布显示精灵矩形；滚轮缩放，**中键拖动**平移。
- **右侧**：编辑选中精灵的名称与矩形数值。
- **状态栏**：提示与当前缩放。

## 日常使用

1. **新建工程**：菜单或工具栏「新建」。
2. **导入精灵表**：菜单 **文件 → 导入精灵表…** 或工具栏「导入精灵表」，选择 PNG/JPEG/WebP 等；底图显示在画布上。若工程里还没有精灵，会自动生成覆盖整张图的一个选区。
3. **添加精灵**：工具栏「添加精灵」，在列表与画布中出现默认矩形。
4. **调整**：在右侧修改宽高与位置，或在画布上拖动选区（有底图时会限制在图内）。
5. **导出**：菜单或工具栏「导出图集」，下载 `atlas.png` 与 `atlas.json`（有底图时按选区从原图切片再打包）。

## 常见问题

- **无法导出**：确保至少存在一个精灵；查看状态栏提示。
- **缩放异常**：使用视图菜单中的「适应窗口」或重置缩放。
