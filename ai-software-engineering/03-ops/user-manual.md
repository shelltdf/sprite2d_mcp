# 用户说明书：Sprite2D 精灵表编译工具

## 环境前提

- 现代浏览器（Chromium / Edge / Firefox 近期版本）。
- 通过开发服务器或构建后的静态文件打开应用。

## 界面说明

- **菜单栏**：文件操作（含**另存工程**、导入/导出第三方图集与**仅元数据**描述）、精灵图/精灵/动画、视图缩放；**主题**（跟随系统 / 浅色 / 深色）、**语言**（中文 / English）；**工具栏**：同等快捷入口（**仅图标**，鼠标悬停可看说明）。
- **左侧 Dock**：折叠块依次为精灵表缩略图与设置、精灵对象表、动画序列列表；单击可选中，**双击精灵行**可将该精灵居中到视口。
- **中央**：上方画布显示精灵矩形与底图；滚轮缩放，**中键拖动**或**空格+左键**平移。
- **中央下方**：动画时间线（条带与播放，与右侧动画预览相互独立）。
- **右侧**：属性区——选中精灵时编辑名称、矩形、锚点、内边界与裁剪预览；选中动画时编辑帧与时长，并可播放动画预览。
- **状态栏**：提示与当前缩放。

## 日常使用

1. **新建工程**：菜单或工具栏「新建」。
2. **设置精灵表图片**：菜单 **文件 → 设置精灵表图片…** 或工具栏同名项，选择 PNG/JPEG/WebP 等；仅加载底图，**不会**自动创建精灵，需自行「添加精灵」或在画布中框选区域。
3. **添加精灵**：工具栏「添加精灵」，在列表与画布中出现默认矩形。
4. **调整**：在右侧修改宽高与位置，或在画布上拖动选区（有底图时会限制在图内）。
5. **导出**：菜单或工具栏「导出第三方图集」，下载 `atlas.png` 与 `atlas.json`（有底图时按选区从原图切片再打包）。
6. **保存工程**：菜单 **文件 → 保存工程…**，得到 `.spriteproj.json`；其中记录精灵与动画的完整数据，精灵表仅保存**路径/文件名与尺寸**，不内嵌图片。下次 **打开工程** 后若提示重新关联精灵表，请再执行「设置精灵表图片…」并选同一文件。**另存工程…** 可先输入新的工程名称再下载（适合改名另存一份）。
7. **仅描述、不要像素**：菜单 **文件 → 导出第三方图集描述（JSON/XML）…**，得到供外部引擎使用的元数据文件（路径、矩形、锚点、动画等），不含任何位图。
8. **导入第三方图集**：菜单 **文件 → 导入第三方图集…** 时，第一步可选 **JSON 或 XML**（如 TexturePacker、Phaser 3、Starling XML、本工具导出的描述等），第二步选择对应的精灵表图片；若格式无法识别，状态栏会提示。
9. **动画预览**：在左侧选中动画序列后，右侧属性区底部可 **播放预览**，按各帧时长轮播精灵表上的切片（与下方时间线播放互不干扰）。

## 常见问题

- **无法导出打包图集**：菜单「导出第三方图集」需至少存在一个精灵；查看状态栏提示。（「导出第三方图集描述」可无精灵，用于空模板或仅记录路径。）
- **缩放异常**：使用视图菜单中的「适应窗口」或重置缩放。
